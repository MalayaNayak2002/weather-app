# 🌦️ Weather App

A responsive weather application that lets users search for any city worldwide and view real-time weather data, including temperature, humidity, and wind speed.

## 📌 Features

- 🔍 Search weather by city name (any city worldwide)
- 🌡️ View current temperature, humidity, and wind speed
- ⚡ Real-time data fetched dynamically from a live weather API
- 🎨 Simple, clean, and user-friendly interface
- 🛡️ Graceful error handling for invalid city names or failed requests

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- REST API (OpenWeatherMap)

## 🏗️ Implementation Highlights

- Asynchronous JavaScript (Fetch API) to retrieve live weather data
- Dynamic DOM updates based on user input, without page reloads
- Clean separation of UI logic and API calls
- Responsive design that works across screen sizes

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/MalayaNayak2002/weather-app.git
   ```
2. Navigate into the project folder:
   ```bash
   cd weather-app
   ```
3. Open `index.html` in your browser

   *(If the project uses an API key, add your own OpenWeatherMap API key in the relevant JavaScript file before running)*
